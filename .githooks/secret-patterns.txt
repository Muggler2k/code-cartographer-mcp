# Secret / PII detection patterns — single source of truth.
# Format: <label>|<regex>   (only the FIRST "|" separates label from regex,
# so regexes may contain "|"). Lines that are blank or start with "#" are ignored.
# Consumed by .githooks/pre-commit (staged-file scan) and
# .githooks/secret-scan-hook.sh (Claude Code PreToolUse write-time scan).
# Tuned to favor specificity over recall to limit false positives on a content repo.
AWS Access Key ID|AKIA[0-9A-Z]{16}
AWS Secret Access Key|aws(.{0,20})?(secret|sk)[^a-z0-9]{0,5}[A-Za-z0-9/+=]{40}
Google API Key|AIza[0-9A-Za-z_\-]{35}
Stripe Live Key|sk_live_[0-9a-zA-Z]{24,}
Stripe Restricted Key|rk_live_[0-9a-zA-Z]{24,}
GitHub Token|gh[pousr]_[A-Za-z0-9_]{36,}
Slack Token|xox[abprs]-[A-Za-z0-9-]{10,}
Private Key Block|-----BEGIN (RSA|DSA|EC|OPENSSH|PGP|ENCRYPTED) PRIVATE KEY-----
JWT Token|eyJ[A-Za-z0-9_-]{10,}\.eyJ[A-Za-z0-9_-]{10,}\.[A-Za-z0-9_-]{10,}
Generic Secret Assignment|(password|passwd|pwd|secret|api[_-]?key|access[_-]?token|auth[_-]?token)[[:space:]]*[:=][[:space:]]*["'][^"']{6,}["']
Social Security Number|(^|[^0-9])[0-9]{3}-[0-9]{2}-[0-9]{4}([^0-9]|$)
Credit Card (Visa/MC/Amex/Disc)|(^|[^0-9])(4[0-9]{12}(?:[0-9]{3})?|5[1-5][0-9]{14}|3[47][0-9]{13}|6(?:011|5[0-9]{2})[0-9]{12})([^0-9]|$)
Driver License Label|(driver[\s_-]?license|DL[\s#:]+[A-Z0-9]{6,})
Bank Routing/Account Label|(routing[\s_-]?(number|#)|account[\s_-]?(number|#))[\s:=]+[0-9]{6,}
IBAN|\b[A-Z]{2}[0-9]{2}[A-Z0-9]{11,30}\b
